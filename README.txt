## jasper repport, spring boot, Direct print, Generate pdf file ###

1- to list all printer installed
   http://localhost/api/printers 
   
2 - to show the list of item
   http://localhost/api/person
   
3 - to generate the pdf file, the file will be store in C:User/Username/Desktop/file.pdf
   http://localhost/api/listpdf
   
   
4 - to print to a real printer
   http://localhost/api/direct-print?printer=YOUR PRINTER NAME
